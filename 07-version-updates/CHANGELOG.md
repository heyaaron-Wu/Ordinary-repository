## v1.0.19 - 2026-03-27

### ✨ 新增功能
- 🤖 **多 Agent 架构** - 创建 worker-agent 专门处理耗时任务

  - worker-agent 负责 Cron 任务和后台处理

- 📋 **路由规则配置** - 新增 `~/.openclaw/routing.json`

  - 修复 shebang：`#!/usr/bin/python3.11`

  - 腾讯 API + AKShare 双数据源
