## 🤖 执行流程

### 日终复盘任务 (22:00)

```
22:00 → Cron 触发
  ↓
1. 交易日检查 (is_trading_day.py)
  ↓
2. 获取市场数据 (AKShare, 8 秒超时)
  ↓
3. 生成复盘报告 (auto_review_automation.py)
   - 市场表现（上证/创业板/科创 50/半导体/新能源）
   - 亏损原因分析（4 层归因）
   - 板块后市看法（动态调整）
  ↓
4. 更新 state.json 和 ledger.jsonl
  ↓
5. Git 提交并推送到 GitHub
  ↓
6. 飞书通知推送完成
```

**执行时长:** 约 10 分钟  
**失败重试:** 2 次  
**推送渠道:** 飞书群机器人

---

### 周报复盘任务 (周五 23:00)

```
周五 23:00 → Cron 触发
  ↓
1. 加载状态文件 (state.json)
  ↓
2. 加载交易账本 (ledger.jsonl)
  ↓
3. 计算本周统计
   - 交易笔数
   - 买卖金额
   - 净流入
  ↓
4. 生成周报 (weekly_report.py)
   - 本周表现
   - 持仓分析（含占比）
   - 亏损原因分析
   - 板块后市看法
   - 下周计划
  ↓
5. 保存周报 (08-fund-daily-review/weekly/)
  ↓
6. 飞书推送摘要
  ↓
7. Git 提交并推送到 GitHub (--push-github)
```

**执行时长:** 约 5 分钟  
**失败重试:** 2 次  
**推送渠道:** 飞书群 + GitHub

## ⚙️ Cron 配置

**配置文件:** `~/.openclaw/cron/jobs.json`

**示例配置:**
```json
{
  "name": "fund-2200-review",
  "schedule": {
    "kind": "cron",
    "expr": "0 22 * * 1-5"
  },
  "task": "基金挑战日终复盘（增强版）",
  "script": "/usr/bin/python3.11 /path/to/auto_review_automation.py --base /path/to/workspace",
  "enabled": true,
  "retry": 2,
  "delivery": "feishu",
  "feishu_webhook": "https://open.feishu.cn/open-apis/bot/v2/hook/xxx"
}
```
